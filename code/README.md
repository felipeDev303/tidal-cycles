# Código `.tidal`

Práctica, ejercicios y sets organizados por categoría. Todos los archivos `.tidal` se ejecutan con el plugin de tu editor (VS Code, Pulsar, Emacs, Vim) con SuperCollider + SuperDirt corriendo.

## [`course/`](course/)

Ejercicios del recorrido inicial de aprendizaje.

| Archivo | Tema |
|---|---|
| `cycles.tidal` | Concepto de ciclo |
| `patterns.tidal` | Patrones básicos |
| `mininotion.tidal`, `mininotion2.tidal`, `mininotion-3.tidal` | Mini-notación progresiva |
| `effects.tidal` | Efectos básicos |
| `allthesamples.tidal` | Catálogo de samples disponibles |
| `tidal-cheat-sheet.pdf` | Cheat sheet oficial en PDF |

## [`techniques/`](techniques/)

Snippets enfocados en una técnica concreta.

- `arpeggios.tidal`
- `tech_house.tidal`

## [`music-styles/`](music-styles/)

Sets temáticos por género musical, con notas en markdown del proceso de cada uno.

| Archivo | Notas |
|---|---|
| `dubtechno.tidal`, `dubtechnov2.tidal` | [`DUBTECHNO.md`](music-styles/DUBTECHNO.md) |
| `grime.tidal` | [`GRIME.md`](music-styles/GRIME.md) |
| `perlon.tidal` | [`PERLON.md`](music-styles/PERLON.md) |
| `microhouse.tidal` | (en construcción) |

## [`jamuary/`](jamuary/)

Jams diarias estilo *Jamuary*. Cada archivo es una pieza independiente nacida de la práctica diaria.
