| [02](tidal-cycles/02/Sesion02.md) | Patrones, ciclos y transformaciones |
| [03](tidal-cycles/03/Sesion03.md) | Sonido, samples y efectos |
| [04](tidal-cycles/04/Sesion04.md) | Composición algorítmica y síntesis |
| [05](tidal-cycles/05/Sesion05.md) | Cierre y exhibición — algorave de clase |
